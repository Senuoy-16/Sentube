# Sentube

**Sentube** is a **YouTube clone** built with **React** and **Supabase**. It allows users to upload, watch, and interact with videos, while also managing their watch history and watchlist. The platform includes user authentication and advanced features like video suggestions and engagement tracking.  

---

## Features

- ✅ User authentication (signup / login)  
- ✅ Upload videos  
- ✅ Watch videos  
- ✅ Like / dislike videos  
- ✅ Video suggestions after selecting a video  
- ✅ Video view tracking (mark as “viewed” after 5 seconds and add to history)  
- ✅ Add videos to personal watchlist  
- ✅ Comments on videos  
- ✅ Video management (add/delete your videos, track likes, dislikes, and comments)  

---

## Tech Stack

- **Frontend:** React, React Router  
- **Backend / Database:** Supabase  
- **Styling / UI:** Material UI

---

## Installation / Setup

1. Clone the repository:

```
git clone https://github.com/Senuoy-16/Sentube.git
cd Sentube
```

2. Install dependencies:

```
npm install
```

2. Create a .env file in the root directory and add your Supabase credentials:

  - REACT_APP_SUPABASE_URL=your_supabase_url
  - REACT_APP_SUPABASE_ANON_KEY=your_supabase_anon_key


2. Start the development server:

```
npm start
```
